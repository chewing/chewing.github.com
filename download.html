--- title: Download ---

下載

各專案釋出的檔案可在 Github 的專案頁面下載。 例如 libchewing。

微軟視窗環境

Windows Chewing（Windows 98 / 2000 / XP）

Windows Chewing TSF（Windows Vista / 7 / 8 / 10）

PIME Chewing（Windows Vista / 7 / 8 / 10）

開源作業系統

現今的主流發行版大多採用 IBus 輸入法框架，如果有收錄進套件庫的話，請安裝 ibus-chewing 即可。

Ubuntu PPA: chewing daily build ppa

編譯

以 Debian GNU/Linux 為例：

  • build-essential : C/C++ 開發環境
  • autoconf
  • automake
  • libtool
  • check : unit testing framework for C

其它還需要以下套件：

  • libscim-dev : SCIM-chewing 所需
  • iiimf-dev : IIIMF-chewing 所需

套件

  • Debian GNU/Linux 已正式收錄 libchewing 與 SCIM-chewing，也可取得最新的非正式套件： APT /etc/apt/sources.list
    deb http://apt.debian.org.tw unstable main
  • SuSE (Official)
  • Fedora 已收錄 libchewing 與 SCIM-chewing
  • Arch Linux 已收錄 libchewing 與 SCIM-chewing
  • FreeBSD 已收錄於 ports，可參見 FreeBSD chinese：libchewing / SCIM-chewing
  • NetBSD 已收錄於 pkgsrc，可參見 SCIM-chewing
  • OpenBSD 已收錄於 ports，可參見 libchewing 與 SCIM-chewing
