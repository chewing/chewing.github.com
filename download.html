下載

各專案釋出的檔案可在 Github 的專案頁面下載。 例如 libchewing。

微軟視窗環境

Windows Chewing TSF（Windows Vista / 7 / 8 / 10 / 11）

PIME Chewing（Windows Vista / 7 / 8 / 10 / 11）

舊版微軟視窗環境

Windows Chewing（Windows 98 / 2000 / XP）

開放原始碼的作業系統

現今的主流 Linux 發行版及 FreeBSD/NetBSD/OpenBSD 大多採用 IBus 或 Fcitx 輸入法框架，如果有收錄進套件庫的話，請安裝 ibus-chewing 或 fcitx5-chewing 即可。

從原始程式碼自行編譯

請參閱以下 GitHub 專案頁面，以得知如何從原始程式碼自行編譯及其前置作業：

  • libchewing: 主體函式庫
  • 各式橋接作業平台的實作
